# FastAPI: Краткое руководство

## I. Основы

* **FastAPI:** Современный, высокопроизводительный веб-фреймворк Python для создания API.
* **ASGI (Asynchronous Server Gateway Interface):** Стандарт для асинхронных веб-приложений в Python.
* **Pydantic:** Библиотека для валидации данных и настройки. Используется FastAPI для определения моделей данных.
* **Type Hints:** Использование аннотаций типов Python для улучшения читаемости и валидации.
* **Dependency Injection:** Механизм для управления зависимостями в коде. FastAPI предоставляет встроенную поддержку.
* **OpenAPI (Swagger UI):** Стандарт для описания API. FastAPI автоматически генерирует документацию OpenAPI.

## II. Основные Концепции

* **Path Operations:** Функции Python, которые обрабатывают HTTP-запросы к определенным путям (endpoints).
* **HTTP Methods:** GET, POST, PUT, DELETE, PATCH – определяют тип операции, которую нужно выполнить.
* **Path Parameters:** Часть пути, которая принимает значения, передаваемые в функцию.
* **Query Parameters:** Параметры, передаваемые в URL после знака `?`.
* **Request Body:** Данные, отправленные в теле HTTP-запроса (например, JSON).
* **Response Body:** Данные, отправленные в теле HTTP-ответа.
* **Status Codes:** Числовые коды, указывающие на результат операции (например, 200 OK, 400 Bad Request).

## III. Основные Типы Данных Pydantic

* **`str`:** Строка.
* **`int`:** Целое число.
* **`float`:** Число с плавающей точкой.
* **`bool`:** Логическое значение (True или False).
* **`list`:** Список.
* **`dict`:** Словарь.
* **`Optional[T]`:** Тип, который может быть `T` или `None`.
* **`Union[T1, T2, ...]`:** Тип, который может быть одним из указанных типов.
* **`datetime`:** Дата и время.

## IV. Основные Декораторы FastAPI

* **`@app.get("/")`:** Определяет функцию для обработки GET-запросов к корневому пути.
* **`@app.post("/items/")`:** Определяет функцию для обработки POST-запросов к пути `/items/`.
* **`@app.put("/items/{item_id}")`:** Определяет функцию для обработки PUT-запросов к пути `/items/{item_id}`.
* **`@app.delete("/items/{item_id}")`:** Определяет функцию для обработки DELETE-запросов к пути `/items/{item_id}`.
* **`@app.get("/items/{item_id}/nested")`:** Пример вложенного пути.

## V. Dependency Injection

* **Dependency:** Любая функция, класс или переменная, необходимая для выполнения другой функции.
* **Dependency Injection:** Предоставление зависимостей функции вместо ее создания или поиска.
* **`def get_db()`:** Пример функции зависимости для получения соединения с базой данных.
* **`def read_item(item_id: int, db: Database = Depends(get_db))`:** Пример использования `Depends` для внедрения зависимости.

## VI. Основные Функции и Классы

* **`FastAPI()`:** Создает экземпляр приложения FastAPI.
* **`IncludeRouter()`:** Включает роутеры в приложение.
* **`HTTPException()`:** Генерирует HTTP-ошибки.
* **`Status`:** Перечисление для представления HTTP-статусов.
* **`Response`:** Класс для настройки HTTP-ответов.

## VII. Асинхронное Программирование

* **`async def`:** Определяет асинхронную функцию.
* **`await`:** Используется для ожидания результата асинхронной операции.
* **`asyncio`:** Библиотека Python для асинхронного программирования.
* **Асинхронные модели Pydantic:** Используйте `Async` для асинхронных операций с моделями.

## VIII. Безопасность

* **OAuth2:** Протокол авторизации.
* **JWT (JSON Web Token):** Стандарт для передачи информации в виде JSON.
* **CORS (Cross-Origin Resource Sharing):** Механизм для разрешения запросов от других доменов.

## IX. Развертывание

* **Uvicorn:** ASGI-сервер для развертывания приложений FastAPI.
* **Docker:** Используйте Docker для контейнеризации приложения.
* **Cloud Providers (AWS, Google Cloud, Azure):** Развертывание на облачных платформах.
